🔫 The Family Records — Mafia Stats Tracker
A browser-based app for tracking your Mafia game group's statistics. Supports multiple game modes with structured role assignment, automatic scoring, and shared cloud data via Supabase.

Features

5 Game Modes: ShouMuRen, Normal (12P), Normal (9P), White Wolf, Jupiter
Structured role assignment — every role slot is listed; assign a player to each one
Side-based win resolution — pick which side won; all players scored automatically
Watchers can be added to any game for +4 pts
Leaderboard sorted by total score with win rate bars
Game History with full role/side/outcome breakdown
Supabase sync — one shared database for your whole group
